## AQUATHON-APP

### Frontend

### Setup Instructions
1. **Go to folder**
     ```sh
    cd aquathon_frontend
    ```
2. **Install Dependencies:**
    ```sh
    pnpm install
    ```

4. **Run the App:**
    ```sh
    pnpm dev
    ```

### Backend
### Setup Instructions

4. **Run the App:**
    ```sh
    pnpm dev
    ```
